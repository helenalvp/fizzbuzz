The Code for FizzBuzz

                        
                            function getValues(){
                                let fizzValue = document.getElementById("fizzValue").value;
                                let buzzValue = document.getElementById("buzzValue").value;
                            
                                fizzValue = parseInt(fizzValue)
                                buzzValue = parseInt(buzzValue)
                            
                                if(Number.isInteger(fizzValue) && Number.isInteger(buzzValue)){
                                    let values = generateValues(fizzValue, buzzValue)
                                    displayValues(values)
                                } else{
                                    alert("Please Enter Integers")
                                }
                            }
                            
                            function generateValues(fizz, buzz){
                                let returnArray = []
                            
                                for(let i = 1; i <=100 ; i++){
                            
                                    let value = ((i % fizz == 0 ? "Fizz": "") + (i % buzz == 0 ? 'Buzz' : "") || i)
                                    returnArray.push(value)
                                }
                                return  returnArray
                            }
                            
                            function displayValues(valArr){
                            
                                let tableBody = document.getElementById("results");
                                let templateRow = document.getElementById("fizzBuzzTemplate");
                            
                                tableBody.innerHTML ="";
                            
                                for(let i = 0; i < valArr.length; i+= 5){
                            
                                    let tableRow = document.importNode(templateRow.content, true)
                                    let rowCols = tableRow.querySelectorAll("td");
                                    
                                    rowCols[0].classList.add(valArr[i])
                                    rowCols[0].textContent = valArr[i]
                            
                                    rowCols[1].classList.add(valArr[i+1])
                                    rowCols[1].textContent = valArr[i+1]
                            
                                    rowCols[2].classList.add(valArr[i+2])
                                    rowCols[2].textContent = valArr[i+2]
                            
                                    rowCols[3].classList.add(valArr[i+3])
                                    rowCols[3].textContent = valArr[i+3]
                            
                                    rowCols[4].classList.add(valArr[i+4])
                                    rowCols[4].textContent = valArr[i+4]
                            
                                    tableBody.appendChild(tableRow)
                                }
                            }

This code prints the numbers from 1 to 100 and uses your fizz and buzz values to change their multiples to "Fizz", "Buzz", or "FizzBuzz" (if both are a multiple of the same integer)

getValues

getValues is a function that gets the user's input from the DOM and changes it to a number. If the values are not numbers, it will prompt the user to input numbers.

generateValues

generateValues is a function that loops through numbers 1 and 100. If a number is a multiple of the user's fizz value, it will be changed to "Fizz". If the number is a multiple of the user's buzz value, it will be changed to "Buzz". If it's a multiple of both fizz and buzz, it will display the value "FizzBuzz". Otherwise, it will display the number.

displayValues

displayValues is a function that displays the new FizzBuzz array from generateValues and adds it to the DOM, using a template that sets five values in a row. It also adds the value of the item in the class, to help shape the css and make the "Fizz", "Buzz", and "FizzBuzz" stand out from the numbers.
